### Create React Project using Vite
```
npm create vite@latest
```
### Install Npm
```
npm install
```
### Run Project using NPM
```
npm run dev
```
### Tailwind Css
```
cd frontend/
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
### tailwind.config.js
```
 content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
```
### src/index.css
```
@tailwind base;
@tailwind components;
@tailwind utilities;
```
